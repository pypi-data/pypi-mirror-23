1. Introduction

This user manual explains how to install and use SimulatorToFMU. SimulatorToFMU is a software package written in Python which allows users to export the distribution simulation program Simulator version 7.2 as a Functional Mock-up Unit(FMU) for model exchange using the Functional Mock-up Interface (FMI) standard version 2.0. This FMU can then be imported into a variety of simulation programs that support the import of Functional Mock-up Units for model exchange. This capability allows for instance to model the distribution grid in Simulator, export the model as an FMU, import and link the model with a transmission grid developed in a tool such as GridDyn.
