tinyq
=====

|Build| |Coverage| |Pypi version|

A tiny job queue framework.


Install
----------

::

    pip install tinyq


Usage
-------

start redis server ::

    $ redis-server


app.py ::


    from tinyq import Application

    app = Application()


    @app.task()
    def add(m, n):
        return m + n


add jobs ::

    for m in range(10):
        for n in range(3):
            add.delay(m, n)
