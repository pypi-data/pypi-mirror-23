## General Information

Create distribution file:

```
python setup.py bdist_wheel
```

Upload distributions

```
twine upload dist/*
```

## Dependencies

- [Twine](https://github.com/pypa/twine) is a utility for interacting with PyPI, that offers a secure replacement for `setup.py upload`.

    ```
    pip install twine
    ```
