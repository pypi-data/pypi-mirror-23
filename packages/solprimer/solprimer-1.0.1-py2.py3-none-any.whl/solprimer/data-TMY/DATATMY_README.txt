In this folder are stored the source TMY (Typical Meteorological Year) files.


TMY files can be downloaded, for example, from:

EPW datasets (worldwide)

https://energyplus.net/weather


TMY (USA)

http://rredc.nrel.gov/solar/old_data/nsrdb/1991-2005/tmy3/

http://rredc.nrel.gov/solar/old_data/nsrdb/1991-2005/tmy3/by_state_and_city.html

http://rredc.nrel.gov/solar/old_data/nsrdb/1991-2005/tmy3/by_state_and_city_old.html


Additional sources are indicated in the 'solprimer' document.

The downloaded files need to be converted with scripts (provided under the main directory) to a common format for all other scripts.


The directory definition can be changed in the script code.
