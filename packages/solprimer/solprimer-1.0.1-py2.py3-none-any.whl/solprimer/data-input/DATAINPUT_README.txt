In this folder are stored the input files required by some of the scripts.


The original file with the solar spectral data used by 'spectrum_AM0AM15.py' is located at

www.pveducation.org/sites/default/files/PVCDROM/Appendices/AM0AM1_5.xls


The .csv libraries with the solar component databases are found at

https://sam.nrel.gov/libraries


Alternatively, comprehensive datasets are delivered together with the SAM (System Advisory Model), a software that needs to be installed. See https://sam.nrel.gov/


Additional information on data sources is found in the 'solprimer' main document.

The directory definition can be changed in the script code.
